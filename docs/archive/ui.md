## Typography System

### Font Stack

```css
/* Primary Font - Brand Identity */
--font-primary: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;

/* Secondary Font - Data/Numbers */

/* Fallback for Indian Languages */
--font-indian: 'Noto Sans Devanagari', 'Noto Sans Tamil', 'Noto Sans Bengali', sans-serif;

/* Fallback Stack */
font-family: var(--font-primary);
font-display: swap; /* Prevent FOIT */
```
